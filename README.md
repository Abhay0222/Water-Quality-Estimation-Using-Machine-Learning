💧 Water Quality Estimation This project predicts and classifies water quality based on various chemical and physical features using multiple supervised machine learning algorithms. The goal is to build reliable models to determine whether water is safe or unsafe for human consumption.

📌 Overview Clean water is essential for life. This project demonstrates how machine learning can be applied to predict water potability by training and testing different classification algorithms, comparing their accuracy, and visualizing the results.

⚙️ Algorithms Tested: Logistic Regression | Decision Tree Classifier | Support Vector Machine (SVM) | K-Nearest Neighbors (KNN) | AdaBoost Classifier | Random Forest Classifier

📂 Dataset The dataset includes chemical and physical properties of water samples, such as: pH Hardness Solids Chloramines Sulfate Conductivity Organic carbon Trihalomethanes Turbidity Potability (Target variable)

✨ Features Clean and preprocess the dataset Perform exploratory data analysis (EDA) Train multiple ML models Evaluate models using: Accuracy Score Confusion Matrix Classification Report Visualize performance comparison of algorithms

🧰 Technologies Stack Python Scikit-learn Pandas NumPy Matplotlib Seaborn
