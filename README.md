# 💧 Water Quality Estimation Using Machine Learning

## 📊 Project Overview

This project predicts and classifies water quality using multiple supervised machine learning algorithms based on various chemical and physical properties of water.

The main objective is to build reliable predictive models that determine whether water is safe or unsafe for human consumption.

---

# 📌 Problem Statement

Clean and safe drinking water is essential for human health. This project demonstrates how machine learning techniques can help analyze water quality parameters and accurately predict water potability.

The project compares the performance of different classification algorithms to identify the most effective model for water quality prediction.

---

# 🎯 Objectives

- Predict water potability using machine learning
- Compare multiple classification algorithms
- Perform data preprocessing and exploratory data analysis (EDA)
- Evaluate model performance using classification metrics
- Visualize and compare algorithm accuracy

---

# 📂 Dataset Features

The dataset contains chemical and physical properties of water samples:

| Feature | Description |
|---|---|
| **pH** | pH value of water |
| **Hardness** | Water hardness level |
| **Solids** | Total dissolved solids |
| **Chloramines** | Chloramine concentration |
| **Sulfate** | Sulfate concentration |
| **Conductivity** | Electrical conductivity of water |
| **Organic Carbon** | Organic carbon content |
| **Trihalomethanes** | Harmful chemical compounds in water |
| **Turbidity** | Clarity level of water |
| **Potability** | Target variable indicating safe/unsafe water |
